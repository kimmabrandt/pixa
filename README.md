# PixaPost
## The Game:
You are a cat who must kill all aliens that have invaded the land as you jump through clouds and past candy canes, collecting all the diamonds.

## Controls:
Left/right arrow - move forward/backward
Up arrow - jump
Spacebar - shoot

## How to Play:
